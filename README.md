</pre>

use it with `/parrot message`

## PONG
The world's worst video rendering system and... a playable version of PONG.
![pong example](pong_example.png)

**Usage**
```
/pong         setup game
/register     register as a player
/start        start game
/u            move paddle up
/d            move paddle down
```
